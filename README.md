### whoami

I research, attack, and teach. Most days that means hunting vulnerabilities in web apps and binaries, building tooling to make the hunt faster, or turning what I learn into interactive labs so the next researcher climbs the same hill quicker.

Computer Engineering Techniques student at **Al-Esraa University** · active on **HackerOne**.

### what I work on

- **Vulnerability labs and education** — interactive, bit-level walkthroughs that follow a vulnerability from input down to silicon.
- **Bug-bounty tooling** — small, sharp utilities that automate the boring parts of recon and discovery.
- **Applied AI** — neural assistants and curated tool directories with a focus on usable interfaces.
- **PoCs and write-ups** — payloads, edge cases, and reproductions worth keeping around.
